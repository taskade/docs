## What's New:

### EVE Can Run a Saved Automation

EVE can now run one of your saved automations on demand. Ask EVE to run a flow by name or id (for example, "run my daily digest now") and it executes the saved manual-trigger flow, waits for the run to complete, and hands back a plain-language summary of the result. Disabled flows are refused with a prompt to enable them first.

See [How Genesis Works](../../../genesis-living-system-builder/genesis/how-genesis-works.md).

***

### EVE Can Build an Automation From a Template

EVE can also create a new automation flow from a plain description. It materializes the flow from a validated template, pinned to the current space. Every flow created this way is **created disabled** — it never fires on its own until you review it and turn it on from the flow editor.

See [How Genesis Works](../../../genesis-living-system-builder/genesis/how-genesis-works.md).
