## What's New:

### Table View Multi-Select and Bulk Delete

Table View now supports multi-select — hold Shift or Cmd/Ctrl and click to select multiple rows at once. Bulk delete lets you remove all selected rows in one action, with full undo support so nothing is lost accidentally.

***

### Public Share URLs for Table View

Table View items can now be shared via public URLs, matching the sharing experience already available in other project views.
