## Improvements & Fixes:

* Added a confirmation dialog when deleting media that is used as agent knowledge.
* Disabling an automation should now terminate all running and scheduled workflows.
* You can now discard your changes made to the app's theme via the themebar with the "Revert" button.
* Added missing descriptions from some toast notifications.
* Fixed an issue where apps with custom domains will be met with a 404 error if `www.` was appended before the url.
* Fixed an issue where new users could not sign up via Google SSO.
* Fixed an issue where after cancelling a media item deletion, the user will be incorrectly redirected to the selected media item.
