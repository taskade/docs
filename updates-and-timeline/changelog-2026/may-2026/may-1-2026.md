## What's New:

### Project List Bulk Delete

You can now select multiple projects in a workspace's project list and delete them in a single action — useful when cleaning up large spaces.

### BYOK (Bring Your Own Key) for Agents — Enterprise

Enterprise workspaces can now supply their own LLM API keys for agent operations. This gives organizations direct control over model access, cost attribution, and compliance without routing credentials through Taskade's shared infrastructure.

### Workspace DNA Embed Support

The Workspace DNA page is now iframe-aware: when embedded in an external page (such as a community detail view), its navigation chrome is hidden so only the DNA content shows.
