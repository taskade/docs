## What's New:

### App Secrets Vault

Genesis apps now have a dedicated Secrets vault for storing sensitive values such as API keys and tokens used by the app at runtime. Secrets are stored encrypted and injected at execution time rather than hardcoded in app logic.

### Outbound Proxy for External API Calls

Genesis apps can now route outbound HTTP requests through Taskade's managed proxy. This allows apps to call external APIs without exposing the origin IP and provides a consistent egress point for access-control allowlisting.

### Publish Blocked on Hardcoded Secrets Detection

Before an app is published, Taskade scans for patterns that look like hardcoded secrets (API keys, tokens, passwords). If any are found, publishing is blocked with a warning, nudging developers to move credentials into the Secrets vault instead.
