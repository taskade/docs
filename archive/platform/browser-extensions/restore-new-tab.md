## Method 2: Reinstall the Extension

If the settings method doesn't work, you can reinstall the extension to reset all preferences.

### Steps:

1. **Remove the Current Extension**
   - Right-click the Taskade logo in your browser's toolbar
   - **Chrome**: Select **Remove from Chrome...**
   - **Firefox**: Select **Remove Extension**
   - **Edge**: Select **Remove from Microsoft Edge**
   - Confirm the removal when prompted

2. **Reinstall the Extension**
   - Visit [taskade.com/downloads](https://taskade.com/downloads)
   - Click the download link for your browser
   - Follow the installation prompts

3. **Optional: Install Alternative New Tab Extension**
   - If you prefer a different new tab experience, you can install another extension after reinstalling Taskade
   - Search for "new tab" extensions in your browser's extension store

## Troubleshooting

### Extension Still Overrides New Tab

If the Taskade extension continues to override your new tab after following these steps:

1. **Check Extension Permissions**
   - Go to your browser's extension settings
   - Ensure Taskade has the necessary permissions
   - Try toggling the extension on/off

2. **Clear Browser Cache**
   - Clear your browser's cache and cookies
   - Restart your browser
   - Check if the new tab has been restored

3. **Check for Conflicting Extensions**
   - Disable other new tab extensions temporarily
   - Test if Taskade's new tab behavior changes

### Browser-Specific Issues

**Chrome Users:**
- Check `chrome://extensions/` for extension conflicts
- Ensure Chrome is updated to the latest version

**Firefox Users:**
- Visit `about:addons` to manage extensions
- Check if Firefox is in private browsing mode

**Edge Users:**
- Go to `edge://extensions/` for extension management
- Verify Edge is running the latest version
